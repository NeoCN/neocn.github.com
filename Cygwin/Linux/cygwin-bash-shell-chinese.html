Cygwin Bash Shell 中文显示

Aug 14th, 2009 | Comments

昨天，在XP上安装了Cygwin，移动的网速还是不错的，很快就安装完成了，只是，发现中文全部显示为问号，而且，ls和dir命令，也没有对目录等的进行着色，以下是解决办法：

1、解决中文文件和目录名显示问题

修改.inputrc文件，打开以下设置： 

set meta-flag on
# 关闭bash命令行8字节字符转义符的转换
set convert-meta off
# 使bash命令行支持8字节字符输入
set input-meta on
# 使bash命令行支持8字节字符输出
set output-meta on

2、修改.bashrc文件中，添加如下代码，解决目录着色问题 

alias ls='ls --color --show-control-chars'
3、修改.bash_profile，增加语言相关环境变量 
export LC_ALL=zh_CN.GBK
export LC_CTYPE=zh_CN.GBK
export LANG=zh_CN.GBK
export XMODIFIERS="@im=Chinput"3
stty cs8 -istrip
stty pass8
export LESSCHARSET=latin1
完成以上修改以后，重新启动Cygwin即可。
